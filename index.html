🔑 Inserisci la tua API Key Groq
Groq è 100% gratuito, velocissimo e non richiede carta di credito!
1. Vai su console.groq.com
2. Registrati gratis con Google o email
3. Clicca su "API Keys" nel menu a sinistra
4. Clicca "Create API Key" e copia la key
5. Incollala qui sotto e clicca Entra
🔒 La key viene salvata solo nel tuo browser e non viene mai inviata ad altri server oltre Groq.
